# Custom Font Files

Place your custom font files here.

## Supported Formats
- `.woff2` (recommended - best compression)
- `.woff` (fallback)
- `.ttf` (if needed)

## File Naming
Name your font files clearly, for example:
- `custom-font-regular.woff2`
- `custom-font-bold.woff2`
- `custom-font-italic.woff2`

## Next Steps
1. Add your font files to this directory
2. Update `app/globals.css` with the correct font file names
3. The font will be automatically available throughout the app
